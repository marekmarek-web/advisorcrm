# PRD – Advisor CRM MVP

## Scope MVP

- **A) Kontakty + Domácnosti + Timeline** – Kontakty, firmy (IČO/DIČ), domácnosti, vztahy. Timeline: call/email/meeting/note + přílohy. GDPR: účely, souhlasy, export dat, výmaz.

- **B) Pipeline + Případy** – Board + list, stavy, pravděpodobnost. Typy: Hypotéka, Investice, Pojištění. Základní pravidla pro úkoly dle stavu.

- **C) Úkoly + Kalendář + Meeting notes** – Události na klienta/případ. Meeting notes: strukturovaný zápis, povinné položky, šablony.

- **D) Dokumenty** – Upload na klienta/případ. Audit: kdo nahrál/stáhl/smazal. Vyhledávání dle názvu/tagu.

- **E) Záznam z jednání + PDF** – Povinná pole, export Client summary PDF, verze.

- **F) Import** – CSV/XLS, šablona, mapování 10 polí, duplikace (email/telefon) + merge.

- **G) Security** – Multi-tenant, RBAC (Admin, Manager, Advisor, Viewer), MFA TOTP, TLS + at-rest, AuditLog.

## Phase 2

Fulltext dokumentů, automatická retence, eIDAS model a UI, rozšíření šablon a workflow.

## Phase 3

Klientský portál, analytika, integrace s externími systémy.
